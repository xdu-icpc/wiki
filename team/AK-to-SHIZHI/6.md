# 补题
## B-Bracelets of Orz Pandas
考场上感觉是个组合数学，但是推了好久式子也没推出来，浪费了好多时间。结束之后，发现这题要用到没学过的知识——杜教筛，也算涨了一波姿势。

其中，（带记忆化的）杜教筛的代码如下：

    ll dj(ll x)
    {
      if(x<=maxp)  return s[x];
      if(d[x])  return d[x];
      ll ans=x*(x+1)/2;
      for(ll i=2,j;i<=x;i=j+1)
      {
        j=x/(x/i);
        ans-=(j-i+1)*dj(x/i);
      }
      return d[x]=ans;
    }
